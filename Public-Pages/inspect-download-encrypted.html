Inspect mode will not inspect downloaded files sent via end-to-end encryption.
Here is a partial list of domains that use end-to-end encryption:
  • web.whatsapp.com
  • web.telegram.org
  • translate.google.com
  • outlook.office365.com
  • outlook.office.com
  • outlook.live.com

Inspect mode will not sanitize downloaded web resources file types, such as: html, css, fonts, javascript, video, audio, images. Those files can still be inspected via 'Page Inspection' policy.
Inspect mode will inspect and sanitize uploaded files from the following domains:
  • mail.google.com
