ZTEdge Release 24.01


New Features
  • Add the ability to use RBI via a ZScaler agent.
  • Platform: Enhanced policies to include an option to enable or disable them.
  • Platform: Added a precanned category policy for the Children's Internet Protection Act (CIPA).
Platform Fixes
  • Fixed an issue related to the Native Application report.
  • Fixed an issue related to applying multiple categories in a category policy.
  • Fixed a block page UI issue for mobile and tablet devices.
Proxy Fixes
  • Fixed several compatibility issues with Webex and Outlook not working properly with PAC file deployments.
  • Fixed an incorrect detection of a Malware on the Outlook logoff page.
  • Fixed a problem causing Google Drive's Active Sync function to fail.
  • Fixed a redirection issue on Gmail.
  • Fixed a problem causing a login issue to the Webex application.
Isolation Fixes
  • Fixed an address book issue with the Outlook Web App.
  • Fixed a copy/paste related issue with the Excel Web App.
  • Fixed a UI issue happening when using the Zoom Web App with VMI.
  • Fixed several UI and small functionality issues with various site using isolation.
System
  • Added several improvements to overall system stability and performance.
