ZTEdge Release 24.02


New Features
  • Platform: Refined the usage of checkboxes when selecting multiple categories in category polices.
  • Platform: Added an option to bring and manage your own certificates when using inspect mode.
Platform Fixes
  • Fixed the link to Certificate Instructions from Ericom's Admin UI.
  • Fixed an IdP configuration issue where underscore wasn't supported in IdP Audience.
  • Fixed an issue with policy import.
Proxy Fixes
  • Fixed several issues causing bad functionality in inspect mode in various websites (cnn.com, nordpass.com, f5.com, inforcloudesuite.me)
  • Fixed an Anti Virus related issue causing false positives when using certain websites.
Isolation Fixes
  • Fixed a CloudFlare authentication issue which caused authentication problems in several websites using CloudFlare.
  • Fixed a problem uploading files to websites using Drag & Drop to perform an upload operation.
  • Fixed a VMI related issue where the camera stopped working after a recent Zoom update.
  • Fixed an issue with email composing on the Outlook web app.
  • Fixed several more issues causing bad functionality in various websites (bbc.com, Kintone, greeneye.co.il, 1337x.to)
System
  • Improved CDR performance in Japan Points of Presence and globally.
