ZTEdge Release 24.04


New Features
  • Isolation (WAI): Add the ability to set up RDP, SSH, or VNC based applications in WAI.
  • Secure Web Gateway: Bring Your Own Certificate - the certificate can now be used in inspection mode.
  • Secure Web Gateway: New policy type allows administrators to block or allow traffic to specific geo-locations.
  • Platform: Built-in IdP users can now be imported into the platform in bulk, using a CSV file.
  • Platform: Block pages can now be customized and branded based on your company's needs.
Platform Fixes
  • Fixed a problem causing an incorrect prioritization of uncategorized security policies.
  • Fixed a problem where DLP Schemas could not be created using Japanese characters.
Isolation Fixes
  • Fixed an issue with WhatsApp web causing the need to re-authenticate every day.
  • Fixed an issue which caused elements to be missing from the print menu, when using isolation.
  • Fixed several UI and functional issues in websites opened in isolation (chatgpt, Atlassian, CNN, and more).
