ZTEdge Release 23.09


New Features
  • Added an ability to allow copying from the protected clipboard to the local machine.
  • A new AI category has been added, which includes Generative AI websites.
  • MSSPs can now monitor the log shipping status for their customers.
  • The ZTEdge platform idle timeout is now configurable.
  • The “Copy Image” option in RBI can now be enabled / disabled.
Fixes
  • Various performance improvements and bugs fixes.
