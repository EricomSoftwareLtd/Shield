ZTEdge Release 23.13


New Features
  • Web Categorization: Multiple categories can now be used in a single policy.
  • Platform: More than one IPSec connections can now be set up for a single PoP.
  • Platform: Reports now contain information about DLP Schemas that were triggered.
Platform Fixes
  • Fixed several security related issues on the Admin UI.
  • Fixed an issue with DLP not working when pasting content from the clipboard.
Proxy Fixes
  • Fixed an issue downloading large files in inspect mode.
  • Fixed several compatibility issues with native applications not working properly with PAC file deployments.
  • Fixed an issue related to the configuration of IPSec connections.
  • Fixed a performance related issue when using DNS over HTTPS (DoH).
  • Fixed an issue with a block page not showing the website's category that caused the block.
Isolation Fixes
  • Fixed an instance in which isolated sessions would freeze.
  • Fixed a sanitization issue related to PDF documents.
  • Fixed an issue with a human verification challange from Cloudflare.
  • Fixed several minor issues on Microsoft 365 applications (Excel & Outlook Online)
System
  • Added several improvements to overall system stability and performance.
