ZTEdge Release 23.11


Platform Fixes
  • Fixed an issue disallowing Administrators from using more than one sanitization URL for CDR.
  • Added a missing Firewall Activity logs report to the ZTNA reports.
Proxy Fixes
  • Fixed an issue with the ICAP service causing it to malfunction.
  • Resolved an issue related to the PAC file deployment that disallowed access to internal applications.
  • Addressed several authentication & login issues related to native desktop applications.
  • Fixed an issue causing incorrect retrieval of SAML groups.
  • Fixed an issue causing an error when downloading a file as part of the sanitization process.
Isolation Fixes
  • Fixed several issues causing incorrect behavior on Rocket Chat.
  • Resolved an issue causing problems when logging in to Jira through Google accounts.
  • Addressed several authentication and login issues related to Google Apps and websites using Google for authentication.
System
  • Added several improvements to overall system stability and performance.
