ZTEdge Release 23.12


New Features
  • RBI browser extension: A browser extension now allows for a seamless file download when using CDR: The file will be downloaded directly as it is being scanned for CDR. The extension is available for download through this link.
  • Improved usage of web categorization in policies: Enhanced the list of web categories to be much more specific and added a search option to quickly find the needed categories.
  • DLP granularity improvements: DLP is no longer a global rule. DLP schemas can now be defined, then associated per policy to comply with the different policies' needs.
Platform Fixes
  • Added firewall activity logs to ZTNA reports.
  • Fixed a problem with the Native Applications and Session History reports.
  • Fixed an issue related to profiles import.
  • Fixed an issue in policies where an incorrect logic in file type restriction caused false-positives.
Proxy Fixes
  • Fixed an issue scanning GZip and BIN files when using Inspect mode.
  • Fixed an authentication related issue causing some native applications to malfunction.
Isolation Fixes
  • Fixed an issue in the InCircle messaging application where old messages would not appear.
  • Fixed a problem logging into Jira using Google Authentication.
  • Fixed a scrolling issue on mobile for WAI users.
  • Fixed several issues around logging into Microsoft web application.
  • Fixed several UI issues that appeared when using the Rocket chat application.
  • Fixed an AdBlocker related issue where ads got blocked while the AdBlocker is disabled.
System
  • Added several improvements to overall system stability and performance.
